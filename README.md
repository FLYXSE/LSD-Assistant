Мощный UserBot для Telegram с множеством функций и модульной архитектурой.

## Что такое UserBot?

**UserBot** — это бот, который работает от имени вашего аккаунта Telegram (в отличие от обычных ботов, которые используют отдельный аккаунт). Он позволяет автоматизировать действия, выполнять команды и расширять функциональность Telegram.

LSD Assistant использует префикс `LSD.` для всех команд (например, `LSD.start`, `LSD.help`).

## Возможности

| Команда | Описание |
|---------|----------|
| `LSD.start` | Приветственное сообщение и меню всех команд |
| `LSD.help` | Справка по использованию бота |
| `LSD.ping` | Проверка задержки отклика |
| `LSD.wiki` | Поиск статей в Wikipedia |
| `LSD.rand` | Генератор случайных чисел или слов |
| `LSD.clear` | Очистка сообщений в чате |

### Разработка

- `LSD.dice` — Отправка кубика (в разработке)
- `LSD.ch` — Отправка сообщений в канал (в разработке)
- `LSD.info` — Информация о пользователе (в разработке)
- `LSD.mute` — Мут пользователя (в разработке)
- `LSD.online` — Проверка онлайн-статуса (в разработке)

## Установка

### 1. Клонирование репозитория

```bash
git clone https://github.com/FLYXSE/LSD-Assistant.git
cd LSD-Assistant
```

### 2. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 3. Настройка переменных окружения

Создайте файл `.env` в корне проекта со следующими данными:

```env
API_ID=your_api_id
API_HASH=your_api_hash
```

**Как получить API_ID и API_HASH:**

1. Перейдите на [my.telegram.org](https://my.telegram.org)
2. Войдите в свой аккаунт
3. Перейдите в раздел "API Development Tools"
4. Создайте новое приложение
5. Скопируйте `api_id` и `api_hash`

### 4. Запуск

```bash
python LSD.py
```

## Использование команд

### LSD.start
Запуск бота и просмотр меню команд.
```
LSD.start
```

### LSD.help
Получение справки.
```
LSD.help
```
Для справки по конкретной команде:
```
LSD.help wiki
```

### LSD.ping
Проверка времени отклика в миллисекундах.
```
LSD.ping
```

### LSD.wiki
Поиск статьи в Wikipedia.
```
LSD.wiki Python
```

### LSD.rand
Генерация случайных данных.

**Случайное число:**
```
LSD.rand +num +1-100
```
- `+num` — режим генерации числа
- `+1-100` — диапазон (от 1 до 100)

**Случайное слово:**
```
LSD.rand +word +letter_f +8
```
- `+word` — режим генерации слова
- `+letter_f` — первая буква слова (f)
- `+8` — длина слова (символов)

### LSD.clear
Очистка сообщений в чате.

**Очистить только свои сообщения:**
```
LSD.clear +me
```
Затем ответьте `LSD.clear.YES` для подтверждения.

**Очистить все сообщения:**
```
LSD.clear +all
```
Затем ответьте `LSD.clear.YES` для подтверждения.

## Требования

- Python 3.10+
- Telegram API ключи (api_id, api_hash)

## Зависимости

- `pyrogram` — основная библиотека для работы с Telegram
- `pyromod` — библиотека для интерактивных диалогов
- `python-dotenv` — работа с переменными окружения
- `wikipedia` — API для Wikipedia

## Важные предупреждения

1. **Правила Telegram**: Использование UserBot может нарушать условия использования Telegram. Используйте на свой риск.
2. **Безопасность**: Никогда не публикуйте свои `API_ID` и `API_HASH`.
3. **Тестовый режим**: Бот запускается в `test_mode=True`, что требует активации через @PyroBot.
